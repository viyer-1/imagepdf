# Changelog

All notable changes to the Image ↔ PDF Converter Open Source Project will be documented in this file.

## [1.1.0-oss] - 2026-05-08

### Changed
- **Open Source Release**: Initial release as an open-source project.

## [1.1.0] - 2026-01-14

### Added
- **Thumbnail Grid**: Visual preview of added files.
- **Drag-and-Drop Reordering**: Reorder files visually in the grid.
- **Custom Page Sizes**: Support for precise PDF page dimensions.
- **Full-screen start**: App opens maximized by default.

## [1.0.0] - 2026-01-14

### Added
- **Bidirectional Conversion**: Images → PDF and PDF → Images.
- **Privacy Features**: Automatic EXIF metadata stripping.
- **Multi-Format Support**: JPG, PNG, TIFF, BMP, WEBP.
- **Local API**: Programmatic access via REST endpoints.
- **Cross-Platform Support**: Windows, macOS, and Linux (including ARM).
